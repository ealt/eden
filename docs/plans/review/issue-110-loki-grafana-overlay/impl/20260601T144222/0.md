## Change Summary

- **Default overlay** tails the #109 per-service JSONL bind-mount (`${EDEN_EXPERIMENT_DATA_ROOT}/logs/**/*.jsonl`) read-only via Alloy → Loki; **no docker socket** (lower privilege than the issue's `docker_sd_configs` sketch, which #109 made unnecessary). Loki + Alloy internal-only; Grafana the only host-exposed surface (`${GRAFANA_HOST_PORT:-3000}`). Images pinned: `grafana/loki:3.7.2`, `grafana/alloy:v1.16.1`, `grafana/grafana:12.4.3`.
- **Alloy, not Promtail** (Promtail EOL 2026-03-02). Pipeline: `local.file_match` → `loki.source.file` → `loki.process` (json/timestamp/labels, lifting service/level/experiment_id to labels, record `ts` as entry time via RFC3339Nano) → `loki.write`.
- **Optional infra overlay** (`compose.logging-infra.yaml`) layered on top adds postgres/forgejo **stdout** capture via Alloy's `discovery.docker` + `loki.source.docker`. It mounts the docker socket, so it is isolated in its own overlay (AGENTS.md privilege-isolation discipline) and uses a dedicated `:?`-guarded `EDEN_LOGGING_DOCKER_GID` (NOT `EDEN_DOCKER_GID`, which defaults to 0).
- **Storage classification:** `loki/` + `alloy/` data-root dirs are DERIVED / non-protocol-owned (chapter 01 §13) — a queryable projection of `logs/`, not durable state.
- **setup-experiment** always generates `EDEN_GRAFANA_ADMIN_PASSWORD` (idempotent) + creates `loki/`+`alloy/` dirs (chmod 0777; Loki uid 10001). No `--with-logging` flag.
- **smoke-logging.sh** (bash 3.2-safe) + `compose-smoke-logging` CI job (path-gated, non-required). Validated end-to-end on a live bring-up incl. the infra overlay.

## Implementation notes / deliberate plan deviations (already decided)

- **Healthchecks:** `grafana/loki:3.x` is distroless (no shell) and `grafana/alloy` ships no HTTP client, so only Grafana (has curl) carries a CMD healthcheck; loki+alloy use `service_started` ordering. The smoke validates Loki readiness/ingestion by exec-ing curl through the Grafana container. (Plan anticipated: "fall back to the image's own health convention otherwise.")
- **Image tags:** plan sketched Grafana 11.x (now EOL); pinned current-stable 12.4.3 per the plan's "exact tags at impl time" instruction.
