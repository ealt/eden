**Overall Assessment**

The plan has the right target cells and the auth-disabled skepticism is directionally correct, but it is not implementation-ready yet. The main problem is that it overstates how much it can inherit from `#147`: as written, the exec-mode overlays do not cover the exp-2 host trio, the shared library breaks the chaos drill by moving it after quiescence, and the auth-disabled branch is much larger than the plan admits.
