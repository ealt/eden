4. **Gate script semantics** — `scripts/check-complexity.py` handles preamble walks, file-level vs function-level scope, radon's per-file CC output shape. False-positive / false-negative risk.

6. **Structural erosion (the dimension this very audit targets)** — Did the refactors themselves introduce new slop? Helpers that are too thin? Duplicated extraction logic?
