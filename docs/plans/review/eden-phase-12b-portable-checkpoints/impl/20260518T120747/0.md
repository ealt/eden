## Change Summary

- **Spec-level:** A new normative chapter 10 plus targeted amendments to chapters 02 / 07 / 08 / 09. The portable checkpoint format is a tar archive of a directory tree containing a manifest, per-component JSONL files, an `experiment.json`, an `experiment-config.yaml`, a `git bundle`, and content-addressed `artifacts/sha256/<hex>` files. Round-trip equivalence is MUST-strength; recovery probe via `imported_from` is MUST-strength.
- **Authority amendment:** the three §14 endpoints are gated on the LITERAL `admin` principal (per §13.1), not the `admins` group. Rationale: bootstrap-class operations — a fresh receiver has no `admins`-group member registered before its first `import_checkpoint`.
- **eden-checkpoint package:** new workspace member with `CheckpointWriter` / `CheckpointReader` / `extract_checkpoint`; typed errors mapped 1:1 to the chapter-7 §9 vocab; tar-extract uses Python 3.12+ `data` filter for path-traversal safety; content-addressed artifact dedup; `repo_bundle.py` subprocess wrappers pinned to a literal author identity so bundles are reproducible.
- **Store layer:** `export_checkpoint(stream, *, experiment_config, repo_bundle, exporter_info=None)` snapshots inside `_atomic_operation` (transactional snapshot per chapter 10 §6); `import_checkpoint(stream, *, as_experiment_id=None, extract_dir=None)` validates spec_version + target experiment_id + store-must-be-empty BEFORE any write; bulk-inserts every entity in a single atomic commit; sets `imported_from`. Workers imported with sentinel hash so source plaintext never authenticates.
- **Wire layer:** three new FastAPI routes, all admin-gated; new `@app.exception_handler(CheckpointError)`; eden-checkpoint `ExperimentIdMismatch` re-raised through wire-layer `ExperimentIdMismatch` so a single class maps the wire type.
- **Conformance:** 7 new scenario files implementing the 7 chapter-9 §5 groups; receiver-IUT pair fixture so the chapter-10 §11 "store must be empty" precondition is satisfied. Reference impl now `v1+roles+integrator+checkpoints` conformant: 218/218 scenarios pass.
