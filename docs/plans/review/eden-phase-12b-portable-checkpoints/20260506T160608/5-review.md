Those behaviors are now part of the chosen contract, so §6 should name them.

Round 5 gets the plan past alternatives. I would stop at completeness: the core design is now coherent, but the new retry probe and global-endpoint carve-out have not yet been fully propagated into the wire implementation inventory or the test matrix.
