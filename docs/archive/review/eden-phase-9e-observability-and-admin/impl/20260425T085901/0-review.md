Fix by catching `GitError` specifically, classifying the expected-old-SHA mismatch case, and re-raising unexpected git failures.

The happy-path admin flows are well-covered and the core integration looks solid; the targeted admin test suite and the admin e2e test both pass. The main gaps are around failure-mode fidelity and plan adherence: read-path transport handling is missing, git delete failures are over-collapsed, and the task-filter semantics drifted from the design and were codified in tests.
